## O que é

A skill que pega um tema, um estilo de comunicação e um público, escolhe o template viral mais estratégico entre 8 opções, monta o hook certo e entrega o roteiro completo de um vídeo curto (45-60s) pronto para gravar — otimizado para alcance, retenção e compartilhamento.

A entrega tem sempre três partes:
1. Roteiro em tabela (Tempo | Cena | Falas)
2. Explicação das escolhas (template, hook, CTA)
3. 5 títulos virais

<Info>
Esta skill é para **viralização ampla** (alcance, retenção, compartilhamento). Se o objetivo for "comenta uma palavra e eu te mando o material", use a skill `instant-value-reels` — são estratégias distintas.
</Info>

## Quando usar

- "Cria um reels viral", "roteiro pra bombar", "vídeo curto pra viralizar"
- "Ideia de Reels / TikTok / Shorts", "script de vídeo curto"
- Qualquer pedido de vídeo de alcance/retenção — não de captura por comentário

## Como utilizar

**O que você pede:** tema do vídeo + estilo/tom + público-alvo.

Se existir avatar do cliente já mapeado, a ISIS puxa de lá — você não precisa descrever tudo do zero.

**Input mínimo (3 itens):**

| # | Item | Exemplos |
|---|------|---------|
| 1 | Estilo / tom | engraçado, misterioso, direto, sarcástico, inspirador |
| 2 | Tema | produtividade, dicas de IA, moda, lançamento do produto X |
| 3 | Público-alvo | criadores de conteúdo, mulheres 25+, empreendedores |

**O que a ISIS faz:**
1. Escolhe o template viral mais adequado ao tom + tema + público (1 de 8).
2. Escolhe o tipo de hook para os primeiros 3 segundos (1 de 3).
3. Monta a estrutura: HOOK → CORPO (com micro-hooks a cada 2-3s) → CTA.
4. Escreve o roteiro completo em tabela, 45-60s.
5. Explica as escolhas e entrega 5 títulos virais.

O roteiro sai completo de uma vez — sem gating passo a passo. Ajuste é oferecido depois.

**O que você recebe:**
- Cabeçalho com template, hook e CTA escolhidos
- Tabela: Tempo | Cena (visual) | Falas
- Explicação das escolhas estratégicas
- 5 opções de título viral

## Mecânica interna

### Os 8 templates virais

| # | Template | Melhor para |
|---|----------|------------|
| 1 | Problema → Solução → Resultado | Tutoriais, hacks, ferramentas |
| 2 | Adivinhação / Mistério | Suspense até revelação |
| 3 | Lista Rápida / TOP 3 | Conteúdo didático rápido |
| 4 | Transformação / Antes e Depois | Resultados visuais |
| 5 | Confissão / História Pessoal | Conexão emocional |
| 6 | Desafio / Contra-intuitivo | Quebra de padrão |
| 7 | Reação / Comentário Polêmico | Opinião forte, engajamento |
| 8 | Ranking / Julgamento Engraçado | Listas, memes, leveza |

### Os 3 tipos de hook (primeiros 3 segundos)

| Tipo | O que faz |
|------|-----------|
| Choque / Inesperado | Quebra de padrão — para o scroll na força |
| Promessa de Benefício | Diz claramente o que o espectador ganha |
| Curiosidade / Intriga | Abre um gap que só fecha no final |

## Usada em quais workflows

Não há workflow fixo — esta skill é acionada diretamente quando o pedido é um vídeo de alcance. Integra bem com:

- [`/workflows/iterar-criativo`](/workflows/iterar-criativo) — para refinar o roteiro após análise de performance
- `/skills/conteudo-10-angulos` — para escolher qual ângulo abordar no vídeo antes de criar o roteiro

## O que NÃO faz

- Não cria reels de "comenta uma palavra e eu mando o material" → use `instant-value-reels`
- Não escreve legenda do post nem carrossel → use `/skills/criativo-carrossel-html`
- Não produz o vídeo nem sugere edição de corte
- Não cria o avatar do público do zero → use `/skills/persona-neurocomportamental`
