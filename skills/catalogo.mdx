Uma **skill** é uma capacidade encapsulada: uma pasta com `SKILL.md` (frontmatter YAML — nome, descrição, versão, tags — \+ corpo com passo a passo). A ISIS e os subagentes carregam skills sob demanda.

<Info>
  **Fonte:** `sistema/skills/<categoria>/<skill>/SKILL.md`. Esta página é **gerada** do frontmatter de cada skill — quando uma skill muda, regera. Total: **59 skills**.
</Info>

<Note>
  **Adaptação:** os nomes e descrições de skill carregam o contexto da sua operação. Ao instanciar o sistema para outra agência, troque as referências de marca/nicho pelas suas — a estrutura (`SKILL.md` por domínio) é a mesma.
</Note>
