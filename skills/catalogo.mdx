Uma **skill** é uma capacidade encapsulada: uma pasta com `SKILL.md` (frontmatter YAML — nome, descrição, versão, tags — + corpo com passo a passo). A ISIS e os subagentes carregam skills sob demanda. Cada skill abaixo tem sua própria página de operador — clique pra ver "como utilizar".

<Info>**Fonte:** `sistema/skills/<categoria>/<skill>/SKILL.md`. Total: **59 skills** (50 de capacidade + 9 de orquestração `workflow_*`).</Info>
