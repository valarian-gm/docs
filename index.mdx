## O que é o Projeto ISIS™

Um **sistema operacional agêntico** para tocar uma agência: uma camada de IA (a **ISIS**, orquestradora) conectada a um cérebro versionado em Git que liga cliente → oferta → criativo → tráfego → dados → relatório → próxima decisão. O dono para de ser refém da execução manual.

O sistema separa **NEGÓCIO** (quem somos, clientes, projetos) do **MOTOR** (`sistema/` — regras, skills, agentes, integrações). O motor é o que se replica e se vende.
